SmartRoll – Smart Attendance Tracking System

SmartRoll is a secure, automated attendance tracking system developed as our Computer Science graduation project. The system verifies student presence using Wi-Fi subnet validation and device MAC address identification, ensuring attendance can only be recorded when students are physically present in the classroom.

The project integrates a Flutter mobile application, a Flask backend hosted on a Raspberry Pi, an SQLite database, and a React-based instructor dashboard to provide real-time attendance monitoring and management. To improve reliability, SmartRoll also implements a heartbeat mechanism that continuously verifies student presence throughout the class session.

Features 
/
📱 Flutter mobile app for students
/
💻 React web dashboard for instructors
/
🍓 Raspberry Pi 5 local backend (Flask)
/
🌐 Wi-Fi subnet verification
/
🔒 MAC address device validation
/
❤️ Heartbeat monitoring for continuous attendance
/
📊 Real-time attendance logging and reporting
/
🗄️ SQLite database for lightweight local storage
\

SmartRoll was designed to provide a scalable, secure, and practical solution for modern classroom attendance management.
